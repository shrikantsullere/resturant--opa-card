# 🏨 Hospitality Operations SaaS Platform

A comprehensive, premium SaaS ecosystem designed for modern hotels, resorts, and restaurants. This platform bridges the gap between property management and dining operations into a single, unified "Single Pane of Glass" experience.

![Tech Stack](https://img.shields.io/badge/Stack-React--Vite--Tailwind-blue?style=for-the-badge)

## 🌟 Key Features

### 🏨 Hospitality & PMS
* **Room Management**: Visual grid for room tracking and maintenance status.
* **Reservations Lifecycle**: Full booking engine with check-in/out automation.
* **Guest Folios**: Integrated billing system for room, restaurant, and service charges.
* **Concierge Hub**: Real-time guest messaging and service request management.
* **Services & Excursions**: Booking engine for transport, tours, and amenities.

### 🍽️ Restaurant & POS
* **Smart POS**: High-speed billing with size selection and special instructions.
* **Kitchen KDS**: Real-time kitchen display for streamlined order processing.
* **QR Smart Experience**: Context-aware QR system for table and in-room dining.
* **Inventory Control**: Cross-departmental stock management with low-stock alerts.

### 📈 Operations & Analytics
* **Executive Dashboard**: Command center for revenue, occupancy, and kitchen load.
* **Duty Board**: Task management for maintenance, housekeeping, and service.
* **Notifications**: Global real-time alert system for staff and management.

## 👥 Roles Supported
* **Admin/Owner**: Full property control and financial oversight.
- **Reception/Concierge**: Guest relations, billing, and booking management.
- **Waiter/Staff**: Dining service and room service delivery.
- **Chef/KDS**: Kitchen operations and inventory management.
- **Customer/Guest**: Self-service ordering, messaging, and billing.

## 🛠️ Tech Stack
* **Frontend**: React 18, Vite, Tailwind CSS.
* **Icons**: Lucide React.
* **State**: Local Context API with LocalStorage persistence.
* **Design**: Custom glassmorphic, mobile-first SaaS aesthetic.

## 🚀 Getting Started

### Installation
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## 📸 Demo Workflow
1. **Admin Login**: Access the Executive Dashboard for property overview.
2. **Reception**: Manage room reservations and guest check-ins.
3. **Guest QR**: Scan QR to order room service or message concierge.
4. **Kitchen**: Receive and process orders via the KDS.
5. **Billing**: View guest folios and settle charges at checkout.
